# Welcome to islam-daily-app

## To Get started

1. Install dependencies

   `npm install`

2. Start the app

   `npx expo start`

## OUTPUT

In the output, you'll find options to open the app in a

- QR Code will at bash.
- Install [Expo Go] app at your mobile phone.
- Open expo go app and Scan qr code.
- Press to reload app and view latest changes.
